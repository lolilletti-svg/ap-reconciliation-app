⚠️ Chrome bloquea las llamadas a la API desde archivos locales
Estás abriendo la app con el protocolo file:// en Chrome, que bloquea las requests externas por seguridad. Tenés dos opciones:
  • 1 Servidor local (recomendado): Abrí una terminal en la carpeta donde está el HTML y ejecutá:
    python3 -m http.server 8080  luego abrí http://localhost:8080 en Chrome.
  • 2 Usá Firefox: Abrí el mismo archivo HTML en Firefox — no tiene esta restricción y funciona directo.
